========================================================
        SISTEMA ACADEMICO - MECANICOPEPE
========================================================

Descripcion
--------------------------------------------------------
Sistema Academico desarrollado para la gestion de usuarios,
vehiculos y procesos relacionados con el proyecto
"MecanicoPepe".

El proyecto fue desarrollado de manera incremental,
mejorando su estructura, funcionalidades y calidad
en cada avance.

[ AVANCE 1 ]
--------------------------------------------------------

- Se organizaron las vistas principales.

[ AVANCE 2 ]
--------------------------------------------------------

[ AVANCE 3 ]

- Se implemento tolerancia a fallos.

- Se mejoro la estabilidad y mantenibilidad del sistema.
